Organizador de tarefas

Gerenciador de tarefas inteligente com categorias e prioridades.

Organizador de tarefas – Gerenciador de Tarefas Inteligente

O Organizador de tarefas é um aplicativo web criado para facilitar a organização diária do usuário. Ele permite adicionar, editar e excluir tarefas, organizando tudo por categorias e prioridades. O painel de progresso mostra quantas tarefas foram concluídas no dia.

Funcionalidades:
  • Adicionar tarefas com prioridade
  • Marcar como concluída
  • Filtro por categoria
  • Interface responsiva e moderna
Tecnologias usadas:

HTML, CSS, JavaScript e Bootstrap

Localizador de restaurantes

Ferramenta para localizar restaurantes próximos e avaliações.

Localizador de restaurantes – Buscador de Restaurantes

O Localizador de restaurantes permite ao usuário localizar restaurantes próximos usando sua posição atual. Além disso, ele exibe avaliações, tipos de comida e distância.

Funcionalidades:
  • Geolocalização
  • Filtros por tipo de comida
  • Avaliações e notas
  • Design intuitivo e responsivo
Tecnologias usadas:

HTML, CSS, JavaScript, API de Geolocalização

EduConectar

Plataforma moderna de cursos com videoaulas e progresso.

EduConectar – Plataforma de Cursos Online

O EduConectar é uma plataforma com cursos em vídeo, progresso do aluno e um painel intuitivo. Inspirado em grandes plataformas, oferece uma navegação simples e eficiente.

Funcionalidades:
  • Área do aluno
  • Progresso do curso
  • Videoaulas integradas
  • Layout responsivo
Tecnologias usadas:

HTML, CSS, Bootstrap e simulação de API

ClimaPro

Aplicação de previsão do tempo com dados detalhados.

ClimaPro – Previsão do Tempo Avançada

O ClimaPro exibe clima atual, previsão de 7 dias e informações detalhadas como umidade, vento e sensação térmica. Seu visual moderno deixa a navegação agradável.

Funcionalidades:
  • Clima em tempo real
  • Previsão de 7 dias
  • Gráfico de temperatura
  • Modo escuro
Tecnologias usadas:

HTML, CSS, JavaScript e consumo de API
